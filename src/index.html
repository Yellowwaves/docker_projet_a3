Projet Développement Web

Elouan Teissere - Lucas Le Bihan

Dans le cadre de notre projet de troisième année à l'ISEN Yncréa Ouest, nous avons réalisé un site web sur l'accidentologie. Ce site web permet de visualiser les accidents survenus en France en 2009, et de prédire le cluster dans lequel se trouve un accident donné. Pour cela nous avons perfectionné nos notions de big data et d'IA pour arriver à ce résultat.
